# create-torimi

The `npm create torimi` scaffolder. It emits a starter Torimi app from a bundled, version-pinned template, with no network fetch at generation time — `create-torimi@X` always produces a template for the X release train.

```sh
npm create torimi my-app
```

Part of the Torimi/Tsubame lockstep release train — keep every `@hayate/*`, `@tsubame/*`, `@torimi/*`, `torimi`, and `create-torimi` package on the **same version**. Start at the [`torimi`](https://www.npmjs.com/package/torimi) README.

Alpha (0.x): no backward-compatibility guarantees.
