Torimi プライバシーポリシー

最終更新日: 2026年7月11日

本プライバシーポリシー（以下「本ポリシー」）は、Torimi（鳥見）（以下「本アプリ」）における 利用者情報の取り扱いについて定めるものです。本アプリは、開発者がアプリの表示を実機で確認するための 開発者向けプレビューツール（dev-client）であり、一般消費者向けのサービスではありません。

要点

  • 本アプリは、氏名・メールアドレス・電話番号・位置情報・広告 ID などの個人情報を収集しません。
  • アクセス解析やトラッキングを行う SDK は一切組み込んでいません。
  • 本アプリが端末内に保存する情報を、開発者のサーバへ収集・送信することはありません。

  • 提供者: HayateProjects

2. 取り扱う情報

本アプリは、その性質上、以下の情報を取り扱います。いずれも本アプリの機能提供のためにのみ用います。

2.1 端末内にのみ保存する情報

次の情報は、利用者の端末内にローカル保存されるもので、開発者のサーバへ送信・収集されることはありません。

  • Device ID（端末識別子） — 開発セッションを識別するために本アプリが発行する、インストール単位の不透明な識別子です。ハードウェア固有の識別子（Android ID など）ではなく、再インストールにより変化します。
  • 接続先 Dev Server の URL — 利用者が入力した接続先アドレスを、次回以降の接続のために端末内に保存します。

2.2 ネットワーク通信

本アプリは、利用者が指定した接続先（利用者自身の開発サーバ、またはデモ配信エンドポイント）から、 プレビュー対象の JavaScript バンドルを HTTP(S) で取得します。この通信は、接続先サーバに対して、 一般的な Web アクセスと同様の情報（IP アドレス、リクエスト日時、User-Agent 等）を伴う場合があります。 これらは接続先サーバの運用者の管理下にあります。デモ配信エンドポイントは HayateProjects が運用します。

また、本アプリは開発支援のためのログ（Device Log）を、 利用者自身が指定した開発サーバに対してのみ送信することがあります。 デモ配信エンドポイントに対して Device Log を送信することはありません。

2.3 カメラの利用（QR コード読み取り）

接続先 URL を入力する手段として、QR コードの読み取り機能を提供します。 読み取りには Google Play services のコードスキャナ（ML Kit）を利用します。 カメラ映像は端末上でオンデバイス処理され、本アプリが画像を保存したり外部へ送信したりすることはありません。 本機能は Google Play services が提供するスキャナ画面が扱うため、本アプリはカメラ権限（CAMERA）を宣言していません。

  • Google プライバシーポリシー

5. 第三者への提供・販売

本アプリは、利用者に関する情報を第三者に販売・提供・共有することはありません。

6. 収集しない情報

本アプリは、氏名、メールアドレス、電話番号、住所、精確な位置情報、連絡先、写真・メディア、 広告識別子（Advertising ID）その他の個人を特定しうる情報を収集しません。

7. 子どもの利用について

本アプリは開発者向けのツールであり、13 歳未満の子どもを対象としていません。 また、子どもから意図的に個人情報を収集することはありません。

8. 本ポリシーの変更

本ポリシーの内容は、法令の変更や本アプリの機能変更に応じて改定することがあります。 重要な変更を行う場合は、本ページ上で改定後の内容を掲示します。
