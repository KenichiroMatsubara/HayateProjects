GPU-native UI Substrate — Element Layer + Raw Layer のデモ集（難易度順）

Canvas Mode (WebGPU + Vello) と HTML Mode (ブラウザ CSS) を自動検出で切り替えます。 WebGPU 対応ブラウザ（Chromium 系）では Canvas Mode が起動します。 非対応環境でも HTML Mode で同一の JS API が動作します。
Lv.1 Raw Layer — 静的描画
ハードコードされた 3 つの矩形を node_create で描画する最もシンプルなデモ。コメント付きで API の全呼び出しを解説。
Raw Layer node_create render 静的
Lv.2 Raw Layer — インタラクティブ
フォームで x / y / w / h / 色 / alpha / radius を指定して矩形を追加・一括削除できる。node_create / node_remove の動作確認。
Raw Layer node_create node_remove インタラクティブ
Lv.3 Element Layer — Hello World
view root + text element を作成し「Hello, Hayate」を描画。テキスト描画の実装状況（Canvas Mode / HTML Mode）をその場で確認できる。
Element Layer element_set_text FONT_SIZE テキスト描画確認
Lv.4 Element Layer — Flex レイアウト
flex コンテナ + 3 つの子ビュー。flex-direction / align-items / justify-content / gap をリアルタイム変更して Taffy レイアウトの動作を確認。
Element Layer DISPLAY.FLEX FLEX_DIRECTION ALIGN_ITEMS GAP
